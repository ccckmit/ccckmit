Diigolet -- Diigo 是一個可以記錄網址並且剪貼網頁內容的網站，Diigolet 則是用來快速記錄網址並剪貼用的快速書籤工具。

翻譯精靈 -- 陳鍾誠設計的書籤工具，可將選取範圍的文章翻譯成中文顯示，這是利用 Google 翻譯 API + JavaScript 所作的書籤小工具。
