# Electron.js 程式範例集

本資料夾共有十幾個範例程式，由淺入深排列！

## 使用方式

1. 請先安裝好 electron.js 的執行環境。
	* 先安裝 node.js -- https://nodejs.org/en/
	* npm install -g electron
2. 想要執行哪個專案，就打下列指令
	* 指令： electron 《專案路徑》
	* 範例： electron 01-hello/a
3. 接著請點選該專案路徑，然後閱讀程式碼。
	* 範例： [01-hello/a](01-hello/a)
